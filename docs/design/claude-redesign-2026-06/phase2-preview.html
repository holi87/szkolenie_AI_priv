Module reading — serif prose + definition callout

Temperatura i powtarzalność

Model językowy nie zwraca jednej „prawdziwej" odpowiedzi — losuje kolejne tokeny z rozkładu prawdopodobieństwa. Parametry generacji decydują, jak bardzo to losowanie jest zachowawcze.

Temperatura (temperature) — skaluje rozkład prawdopodobieństwa. Niska (0,0–0,2) = wyjścia powtarzalne i przewidywalne; wysoka (>1,0) = bardziej różnorodne, lecz trudniejsze do testowania.
